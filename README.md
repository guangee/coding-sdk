# 使用方式

在pom.xml中加入以下代码

```xml
<parent>
<groupId>com.github.guangee.parent</groupId>
<artifactId>parent</artifactId>
<version>版本号</version>
</parent>
```

```xml

<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>


```
